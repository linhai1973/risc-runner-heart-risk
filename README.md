# 跑者心脏风险评估系统 (RISC)

Runner's Integrated Safety & Care - 跑者心脏风险评估系统

## 简介

本系统通过自适应问卷评估跑者的心脏风险指数（0-100分），并提供个性化健康建议。

## 功能特点

- 自适应问卷（13道题，动态跳转）
- RISC指数计算（0-100分）
- 风险百分位显示
- 中医经络急救知识评估
- 西医急救技能评估
- 个性化建议生成
- 管理后台（需登录）
- 飞书云端数据同步
